<h2>Makefile Commands;</h2>
<p>
To run a "make" command, go to the working directory (the one with the Makefile), and type out "make <COMMAND>" replacing <COMMAND> with the command you want or no command to run default <br> 
<br>
-Make commands: <br>
* default ~ runs if given no command, runs the build command <br>
* build ~ compiles the java project into the bin <br>
* run ~ runs the main/Main.class file in the bin directory (runs the application) <br>
* test ~ runs all the junit tests <br>
* docs ~ runs the clearD command, and then builds the javadoc, placing it in the docs directory <br>
* clear ~ runs the clearB command <br>
* clearB ~ deletes the bin directory <br>
* clearD ~ deletes the docs directory
</p>
